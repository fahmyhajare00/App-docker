# FootManager ⚽

FootManager est une application web de gestion des équipes de football. 
Elle permet de gérer les équipes, les joueurs, les informations des matchs et le suivi des performances.

## Fonctionnalités
- Gestion des équipes et des joueurs
- Ajout, modification et suppression des données
- Affichage des informations des équipes
- Calcul et suivi des statistiques
- Interface simple et intuitive

## Technologies utilisées
- Front-end : React.js
- Back-end : Node.js / Express.js
- Base de données : MongoDB
- Conteneurisation : Docker
